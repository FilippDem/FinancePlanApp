# Deprecated PyQt6 Versions

This directory contains archived versions of the Financial Planner application that are no longer actively maintained or used.

## Files

### FinancialPlanner.py
- **Original desktop version** using PyQt6 framework
- **Size**: 2,578 lines
- **Status**: Deprecated - Replaced by web-based Streamlit version
- **Last Used**: Before migration to Streamlit (V15)

### FinancialPlannerV2.py
- **Second iteration** of desktop version using PyQt6 framework
- **Size**: 3,760 lines
- **Status**: Deprecated - Replaced by web-based Streamlit version
- **Last Used**: Before migration to Streamlit (V15)

## Why These Were Archived

These PyQt6 desktop versions were replaced by **FinancialPlannerV15_ClaudeCodeV.py**, which offers:
- Web-based interface (accessible from any browser)
- Better cross-platform compatibility
- No desktop installation required
- Modern UI with Streamlit
- Enhanced features (14 global locations, multi-generational planning, Monte Carlo simulations)
- Easier deployment and updates

## Current Active Application

The current production application is:
- **File**: `FinancialPlannerV15_ClaudeCodeV.py` (located in project root)
- **Framework**: Streamlit (web-based)
- **Launch**: Use `run.sh` (Mac/Linux) or `run.bat` (Windows)
- **Access**: `http://localhost:8501`

## Preservation Purpose

These files are preserved for:
- Historical reference
- Code comparison and learning
- Potential feature recovery if needed
- Documentation of project evolution
